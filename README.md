# 📱 QR Code Scanner App

This project was developed as part of my Android Development Internship at Prodigy Infotech.

## 🚀 Features
- Scan QR codes using device camera  
- Display scanned information instantly  
- Automatically open URLs in browser  
- Simple and user-friendly interface  

## 🛠️ Technologies Used
- Java  
- Android Studio  
- ZXing Library (QR Code Scanning)  

## 📂 Project Details
- Internship: Prodigy Infotech  
- Task Number: PRODIGY_AD_05  
- Application Name: QR Code Scanner  

## 📸 How It Works
1. Click on "Scan QR Code" button  
2. Camera opens to scan QR code  
3. App decodes and displays the result  
4. If the result is a link, it opens automatically in browser
